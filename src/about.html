About

dout.dev is an editorial lab for frontend craft.

The site is written from markdown, rendered through a custom CMS, and shaped around performance, accessibility, and a strong visual point of view instead of a generic theme.

What gets published

CSS systems, accessibility, design choices, build tooling, and the practical reality of maintaining a modern static site by hand.

How it is built

Markdown source files are transformed into static HTML through the in-repo CMS and deployed to GitHub Pages under the custom domain dout.dev.
